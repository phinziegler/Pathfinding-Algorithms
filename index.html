Tutorial

The left panel contains tools for drawing mazes, and determining thier start and end points. The right panel contains tools for manipulating the canvas, and choosing options.

Left Panel

  • Clears all tiles
  • Clear individual tiles
  • Draw maze walls
  • Draw start position(s)
  • Draw end position(s)

Right Panel

  • Zoom out
  • Zoom in
  • Move canvas
  • Play/clear simulation
  • Change animation speed
  • Change algorithm
